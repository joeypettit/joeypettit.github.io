Recent Work

Fecha - Scheduler

This is a full stack scheduling web application that I created using a React-based front end and a Node/Express backend with a PostgreSQL database. It is designed primarily for instructors teaching individual classes through Zoom or Skype. It's based around a calendar-view scheduling interface, which I built from scratch without the use of a third-party calendar library.

Server-Side Calculator

A pocket-style calculator that takes input from the client, runs the calculations on the server before relaying the results back to the client. This was one of my first projects that utilized a backend server. I had some fun with the styling, using vanilla CSS to model the look of a Texas Instruments TI-83 Plus. All functionality and DOM manipulation was implemented with Javascript and jQuery.
